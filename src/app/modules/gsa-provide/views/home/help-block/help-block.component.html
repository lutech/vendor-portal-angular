eOffer/eMod FAS ID Guidelines

Starting March 8, 2021, contractors must use FAS ID multi-factor authentication to access eOffer/eMod. For more information about the implementation of FAS ID for eOffer/eMod, please visit GSA Interact.

Who must Register to use FAS ID for eOffer/eMod?

Offerors/contractors that have not already registered for a FAS ID for another GSA application (FAS Sales Reporting Portal (SRP), GSA Advantage Purchase Order Portal, GSA Vendor Portal, the Mass Mod Portal, eBuy, etc.).

Note for New Offerors: please ensure that you are listed in SAM as one of the following points of contact:

  1. Government Business POC
  2. Electronic Business POC
  3. Government Business Alternate POC
  4. Electronic Business Alternate POC
  5. Past Performance POC
  6. Past Performance Alternate POC

Note for Current Contractors: please ensure that you are listed as an authorized negotiator on the contract (with or without signing authority).

Who does not need to register for a FAS ID to access eOffer/eMod?:

Offerors/contractors that are current eOffer/eMod users and use FAS ID for another application (FAS Sales Reporting Portal (SRP), GSA Advantage Purchase Order Portal, GSA Vendor Portal, the Mass Mod Portal, and eBuy). Your account will be validated based on you being an authorized negotiator on the contract. You do not need to register.
